# XanRunner
Python 3.6 game i've created. To try it out simply download XanRunner.rar and execute XanRunner.exe
#############################################################################################
WINDOWS ONLY
#############################################################################################
The main idea is to avoid Xanax bars, they will get faster each time you don't hit one.
